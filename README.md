# AI Smart Interview Analyzer

AI Smart Interview Analyzer is a Python-based application that simulates technical interviews for different job roles and salary packages. The system asks randomized interview questions, evaluates candidate responses, calculates performance scores, and provides interview feedback such as Selected, Shortlisted, On Hold, or Not Selected.

## Features
- Object-Oriented Programming (OOP) implementation
- Job Role Selection
- Salary Package Selection
- Random Interview Question Generation
- Answer Evaluation
- Performance Scoring
- Interview Result Generation
- Simulated HR Feedback

## Technologies Used
- Python
- Object-Oriented Programming (OOP)
- Random Module
